# chatbot_ui

Vue 3 + Vite chatbot UI. Run as a standalone app or embed the widget on client websites with a single script tag.

[VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Project setup

```sh
npm install
```

## Running the project

### Development (with hot reload)

```sh
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

### Production build (main app)

```sh
npm run build
```

Output: `dist/` (ready to deploy). The app is served from `dist/index.html`.

### Embed build (widget for client sites)

```sh
npm run build:embed
```

Output: `public/embed/chatbot.js` and `public/embed/chatbot_ui.css`. Used so client websites can load the chatbot with one script tag (see [Embed usage](#embed-usage) below).

## Embed usage

Embed the chatbot on any website by loading one script. You host the files; clients add one line to their HTML.

### Structure

| URL | File | Role |
|-----|------|------|
| `https://your-domain.com/chatbot.js` | **public/chatbot.js** | Loader: loads `embed/chatbot.js` |
| (loaded by loader) | **public/embed/chatbot.js** | Embed bundle (Vue + UI), generated by `npm run build:embed` |
| (loaded by bundle) | **public/embed/chatbot_ui.css** | Widget styles |

### What you need to do

1. Build the embed (if not already done):
   ```sh
   npm run build:embed
   ```

2. Build the main app (so `public/` is copied into `dist/`):
   ```sh
   npm run build
   ```

3. Deploy `dist/`. Share with clients: `https://your-domain.com/chatbot.js`.

### What clients add to their site

One line before `</body>` (replace with your URL):

```html
<script src="https://your-domain.com/chatbot.js"></script>
```

The widget appears in the bottom-right corner (button + chat window). CSS is isolated via Shadow DOM so it does not conflict with the host page.

### Test the embed locally

After `npm run build:embed` and `npm run build`:

```sh
npx serve dist
```

Open a page that includes `<script src="/chatbot.js"></script>` to see the widget.
